- ## Psuedocode / Major Steps

  image:

  - import libraries
  - define image path
  - read image

  - destroy all window

  - import libraries
  - define video path

  - set waitkey
  - release memory
  - destroy all window

  - import libraries
  - read / capture video from webcam
  - visualize video by frame
  - write video
  - set waitkey
  - release memory
  - destroy all window
